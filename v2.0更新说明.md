### 2. 🏥 日本医疗术语对照表（核心亮点！）

**位置**：点击底部"指南"标签 → 滚动到"🏥 日本医疗术语对照表"

**包含内容**：

#### 📍 常用医疗机构
| 日本語 | 中文 | English | 罗马音 |
|--------|------|---------|--------|
| 産婦人科 | 妇产科 | OB/GYN | sankafujinka |
| 総合病院 | 综合医院 | General Hospital | sougoubyouin |
| クリニック | 诊所 | Clinic | kurinikku |
| 助産院 | 助产院 | Midwifery Clinic | josanin |

#### 🔬 常见检查项目
- 妊婦健診 (ninpu kenshin) - 产检
- 血液検査 (ketsueki kensa) - 血液检查
- 経口ブドウ糖負荷試験 (OGTT) - 口服葡萄糖耐量试验
- 超音波検査 (chouonpa kensa) - B超

#### 🩸 血糖值相关术语
- 空腹時血糖値 (kuufuku ji kettou chi) - 空腹血糖值 - **目标: <92 mg/dL**
- 食後血糖値 (shokugo kettou chi) - 餐后血糖值 - **目标: 1h<140, 2h<120**
- ヘモグロビンA1c (hemogurobinA1c) - 糖化血红蛋白 - **目标: <5.8%**
- 低血糖 (teikettou) - 低血糖 - **警戒: <70 mg/dL**

#### 💊 治疗方式
- 食事療法 (shokuji ryouhou) - 饮食疗法
- 運動療法 (undou ryouhou) - 运动疗法
- インスリン療法 (insurin ryouhou) - 胰岛素治疗
- 血糖自己測定 (kettou jiko sokutei) - 血糖自我监测

#### 💬 就医常用表达
- 予約をお願いします (yoyaku o onegai shimasu) - 我要预约
- 血糖値が高いです (kettou chi ga takai desu) - 血糖值高了
- 気分が悪いです (kibun ga warui desu) - 我感觉不舒服
- 母子手帳を持ってきました (boshi techou o motte kimashita) - 我带了母子健康手册

#### 📅 日本产检时间表
| 孕周 | 检查频率 | 日语 |
|------|---------|------|
| ~23周 | 每4周一次 | 4週に1回 |
| 24~35周 | 每2周一次 | 2週に1回 |
| 36周~分娩 | 每周一次 | 毎週1回 |

**特别提示**：⚠️ GDM患者可能需要更频繁的检查，请遵医嘱。

## 📊 对比市场同类产品

根据调研，我们发现市场主流产品的问题：

### ❌ 主流产品的痛点
1. **记录负担重** - 需要多次分别填写早中晚餐
2. **硬件依赖** - 必须购买特定品牌血糖仪
3. **过度商业化** - 充满广告和购物链接
4. **收费昂贵** - 医生咨询、高级功能都要钱
5. **数据隐私问题** - 上传到云端，有泄露风险
6. **缺少语言支持** - 单一语言，不适合海外用户

### ✅ 我们的差异化优势

| 维度 | 糖妈日记 v2.0 | 市场产品 |
|------|--------------|---------|
| **目标用户** | 🎯 在日中国准妈妈 | 泛用户 |
| **语言支持** | 🟢 中日英三语 | 🔴 单语言 |
| **医疗术语** | 🟢 日本医学术语对照表 | ❌ 无 |
| **记录方式** | 🟢 一次性输入全天 | 🔴 多次分别输入 |
| **价格** | 🟢 完全免费 | 🔴 部分收费 |
| **硬件要求** | 🟢 无需设备 | 🔴 需购买血糖仪 |
| **数据隐私** | 🟢 本地存储 | 🟡 云端存储 |
| **界面风格** | 🟢 清爽无广告 | 🔴 商业化严重 |

## 🎯 使用场景示例

### 场景1：第一次去日本产婦人科

```
【准备阶段】
1. 打开糖妈日记
2. 点击"指南"→ 滚动到"日本医疗术语对照表"
3. 学习关键术语：
   - 産婦人科 (sankafujinka) - 妇产科
   - 妊婦健診 (ninpu kenshin) - 产检
   - 血液検査 (ketsueki kensa) - 血液检查

【就诊时】
4. 出示母子手帳 (boshi techou)
5. 使用学到的日语：
   "予約をお願いします" (我要预约)
   "血糖値が高いです" (血糖值高了)
```

### 场景2：医生要求看血糖记录

```
【导出数据】
1. 点击"指南" → "数据管理"
2. 点击"📤 导出数据"
3. 将 JSON 文件发给医生

【展示界面】
或者直接把手机给医生看：
1. 点击右上角语言切换器
2. 选择"日本語"
3. 界面立即切换为日语
4. 医生可以直接阅读
```

### 场景3：家人想了解情况

```
【切换到中文】
1. 点击语言切换器 → "中文"
2. 把手机给家人看
3. 所有数据一目了然

【切换到英文】（如果和国际医生交流）
1. 点击语言切换器 → "EN"
2. 所有医学术语自动翻译为英文
```

## ⚠️ 注意事项

### 浏览器兼容性
- ✅ Chrome（推荐）
- ✅ Safari
- ✅ Edge
- ✅ Firefox
- ⚠️ 微信内置浏览器（可能有缓存问题）

### 数据安全
- 所有数据存储在浏览器localStorage
- 清除浏览器缓存会丢失数据
- **建议每周导出备份一次**

### 刷新页面
- 更新已部署，但可能需要强制刷新
- 刷新方法：
  - iOS Safari：下拉刷新
  - Android Chrome：下拉刷新
  - 电脑浏览器：Ctrl+F5 或 Cmd+Shift+R
