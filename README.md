## 🎯 血糖控制目标

基于 ACOG、ADA、中华医学会指南：

| 监测时间 | 目标值 |
|---------|--------|
| 空腹血糖 | < 95 mg/dL (5.3 mmol/L) |
| 餐后1小时 | < 140 mg/dL (7.8 mmol/L) |
| 餐后2小时 | < 120 mg/dL (6.7 mmol/L) |

## 💡 控糖小技巧

### 🍚 碳水控制
- **早餐**：15-30g（血糖最难控制）
- **午餐/晚餐**：45-60g
- **选择低GI主食**：玄米、杂粮 > 白米、白面

### 🥗 进餐顺序
1. **先吃蔬菜**（5-10分钟）
2. **再吃蛋白质**（肉类、豆腐）
3. **最后吃主食**（米饭、面条）

> 这个顺序可以减缓碳水吸收，降低血糖峰值

### 🏃‍♀️ 餐后运动
- **时机**：餐后15-30分钟开始
- **方式**：散步、做家务、轻度活动
- **时长**：20-30分钟
- **强度**：微微出汗，能正常说话

## ⚠️ 重要提示

1. **本应用仅供参考**，不替代医生的专业建议
2. **出现异常**请及时就医：
   - 空腹血糖 > 100 mg/dL
   - 餐后血糖频繁 > 140 mg/dL
   - 出现头晕、心慌等症状
3. **遵医嘱用药**：如需胰岛素治疗，请遵循医生指导
4. **定期产检**：按时进行糖耐量测试和产科检查
