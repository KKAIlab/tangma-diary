- ❌ アカウント登録不要

  - 中国語・英語の対訳

## 📖 使用例

### シーン1：朝の記録

```
【朝起床後】
1. アプリを開く
2. 日付：今日（自動）
3. 体重：67.5 kg
4. 空腹時血糖値：92 mg/dL

【朝食後2時間】
5. 朝食欄：
   - 食事：玄米ご飯、卵、ブロッコリー
   - 食後血糖値：112 mg/dL
   - 測定時刻：08:30

6. 「保存」ボタンをタップ
→ すべてのデータが一度に保存されます
```

### シーン2：1日分の記録を一度に入力

```
【夜、寝る前にまとめて記録】
1. 日付：今日
2. 毎日データ：
   - 体重：67.3 kg
   - 空腹時血糖値：88 mg/dL
   - 歩数：7200歩
   - 備考：体調良好、散歩30分

3. 朝食：
   - 食事：玄米、納豆、味噌汁
   - 血糖値：110 mg/dL
   - 時刻：07:30

4. 昼食：
   - 食事：鶏胸肉、野菜、玄米
   - 血糖値：118 mg/dL
   - 時刻：12:30

5. 夕食：
   - 食事：魚、豆腐、玄米
   - 血糖値：108 mg/dL
   - 時刻：18:30

6. 「保存」ボタンをタップ
→ 1日分のすべてのデータが保存完了！
```

### シーン3：産婦人科での診察時

```
【データを医師に共有】
1. 「ガイド」タブ→「データ出力」
2. JSONファイルがダウンロード
3. 医師にメール送信またはデータを見せる

【日本の医学用語を確認】
1. 「ガイド」タブを開く
2. 「🏥 日本医療術語対照表」セクションをスクロール
3. 必要な用語を確認：
   - 空腹時血糖値 = 空腹時血糖値（くうふくじけっとうち）
   - 食後血糖値 = 食後血糖値（しょくごけっとうち）
   - ヘモグロビンA1c = HbA1c（ヘモグロビンエーワンシー）
```

## 🏥 日本の妊娠糖尿病管理基準

本アプリは日本産科婦人科学会の基準に準拠しています：

| 測定項目 | 目標値 | 備考 |
|---------|--------|------|
| 空腹時血糖値 | < 92 mg/dL | 朝食前（8時間絶食後）|
| 食後1時間血糖値 | < 140 mg/dL | 食事開始から1時間後 |
| 食後2時間血糖値 | < 120 mg/dL | 食事開始から2時間後 |
| HbA1c | < 5.8% (NGSP) | 過去3ヶ月の平均血糖を反映 |

## 💡 機能詳細

### 📅 記録できるデータ

#### 毎日データ
- ⚖️ 体重（kg）
- 🩸 空腹時血糖値（mg/dL）
- 👣 歩数
- 📝 備考（体調、特記事項など）

#### 食事記録
各食事ごとに：
- 🍚 食事内容（複数の食品を入力可）
- 🩸 食後血糖値（mg/dL）
- ⏰ 測定時刻

### 📊 統計機能

- **週間達成率**：朝・昼・夕の血糖値が目標範囲内である割合
- **平均食後血糖値**：すべての食後血糖値の平均
- **血糖値トレンドグラフ**：日々の変化を視覚化
- **食事別分析**：朝・昼・夕ごとの血糖値パターン

### 🏥 日本医療用語対照表

含まれる用語カテゴリ：
1. **医療機関**：産婦人科、総合病院、クリニック
2. **検査項目**：妊婦健診、血液検査、OGTT
3. **血糖値関連**：空腹時血糖値、食後血糖値、HbA1c
4. **治療方法**：食事療法、運動療法、インスリン療法
5. **よく使う表現**：予約、症状の説明など

各用語には以下が付属：
- ✅ 日本語表記
- ✅ ローマ字読み（発音ガイド）
- ✅ 中国語訳
- ✅ 英語訳
- ✅ 補足説明（目標値など）

## ⚠️ 重要事項

### データの保存について
- すべてのデータはブラウザのlocalStorageに保存されます
- ブラウザのキャッシュをクリアするとデータが消えます
- **週1回以上のデータバックアップを推奨**

### バックアップ方法
1. 「ガイド」→「データ管理」
2. 「📤 データ出力」をタップ
3. JSONファイルを安全な場所に保存
   - クラウドストレージ（Google Drive、iCloudなど）
   - メールで自分宛に送信
   - PCにコピー

### データ復元方法
1. 「ガイド」→「データ管理」
2. 「📥 データ取込」をタップ
3. 保存したJSONファイルを選択

## 🆘 よくある質問（FAQ）

### Q1: 会員登録は必要ですか？
**A**: いいえ、会員登録は一切不要です。ブラウザを開けばすぐに使えます。

### Q2: データはどこに保存されますか？
**A**: すべてのデータはお使いの端末（ブラウザのlocalStorage）内に保存されます。サーバーにアップロードされることはありません。

### Q3: 複数の端末で同期できますか？
**A**: データ出力・取込機能を使えば、別の端末にデータを移行できます。ただし、自動同期機能はありません。

### Q4: 日本語が読めなくても使えますか？
**A**: はい、中国語・英語にも対応しています。画面右上の言語切替ボタンで切り替えられます。

### Q5: 産婦人科の先生にデータを見せられますか？
**A**: はい、「データ出力」機能でJSONファイルをダウンロードし、メールで送信したり、画面を直接見せたりできます。

### Q6: オフラインでも使えますか？
**A**: はい、一度アクセスすればオフラインでも使用できます（PWA機能）。

### Q7: iPhoneとAndroidのどちらでも使えますか？
**A**: はい、どちらでも使えます。Chrome、Safari、Edge、Firefoxなどの主要ブラウザに対応しています。

## 🎯 在日中国人妊婦へのメッセージ

このアプリは、**日本で妊娠糖尿病と診断された中国人妊婦の皆さま**のために開発されました。

### 解決する課題

1. **言語の壁**
   - 日本の産婦人科で使われる医学用語が分からない
   - 健診で何を聞かれているか理解できない
   - 検査結果の意味が分からない

2. **記録の負担**
   - 毎日の血糖値測定が面倒
   - 何度も入力するのが大変
   - 続けられない

3. **医師とのコミュニケーション**
   - データをうまく伝えられない
   - 質問したいことを日本語で言えない

### 本アプリの特長

✅ **日本の医療用語対照表**で言葉の壁を解決
✅ **1回の入力で1日分保存**で記録の負担を軽減
✅ **データ出力機能**で医師とスムーズに共有
✅ **完全無料・プライバシー保護**で安心して使用
