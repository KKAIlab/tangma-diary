# 🚀 3分钟快速配置 Google Drive 同步

## 第一步：创建项目（30秒）

1. 打开 https://console.cloud.google.com/
2. 点击顶部项目下拉框 → "新建项目"
3. 项目名称输入：`tangma-diary`
4. 点击"创建"

## 第二步：启用API（30秒）

1. 项目创建后，点击"启用API和服务"
2. 搜索框输入：`drive`
3. 点击"Google Drive API"
4. 点击"启用"

## 第三步：配置OAuth（1分钟）

### 3.1 配置同意屏幕（首次必须）

1. 左侧菜单 → "OAuth同意屏幕"
2. 选择"外部" → 点击"创建"
3. 填写表单：
   - **应用名称**：`糖妈日记`
   - **用户支持电子邮件**：选择您的邮箱
   - **开发者联系信息**：填写您的邮箱
4. 点击"保存并继续"
5. **作用域**页面：直接点击"保存并继续"（不需要添加）
6. **测试用户**页面：
   - 点击"+ ADD USERS"
   - 输入您的Gmail邮箱
   - 点击"添加"
7. 点击"保存并继续"
8. 点击"返回到信息中心"

### 3.2 创建凭据（1分钟）

1. 左侧菜单 → "凭据"
2. 点击顶部"+ 创建凭据" → "OAuth 客户端ID"
3. 应用类型：选择"**Web应用**"
4. 名称：`tangma-diary-web`
5. **已获授权的JavaScript来源**：
   - 点击"+ 添加URI"
   - 输入：`https://kkailab.github.io`
6. **已获授权的重定向URI**：
   - 点击"+ 添加URI"
   - 输入：`https://kkailab.github.io/tangma-diary/`
7. 点击"创建"

### 3.3 复制客户端ID

创建成功后会弹出窗口显示：
```
您的客户端ID
xxxxx-xxxxxxxx.apps.googleusercontent.com
```

**请复制这个客户端ID**，然后告诉我！
